The Gnarly Janitor

by: James Gualtieri (jgualtie)

In a typical day at the skate park, some wild antics can result in some wild messes; luckily, there's someone out there making sure everything is squeaky clean!

Throughout the game, the player walks (or rather skates - sick!) around the level cleaning up messes. After cleaning up so many messes, the player has to grab a clean mop from the supply closet.

Gameplay

The game begins with a level of an assorted arrangement of blood and vomit stains throughout the area. The user must clean the stains by moving over them, resupplying when needed. There is a time limit, and if the user wins, a win screen is displayed, and if the player is a loser a death screen displays.
